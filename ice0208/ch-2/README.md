## Machine Learning

과거의 데이터를 이용하여 미래를 예측하는 기술

### 머신러닝 분류

- Regression (회귀)<지도>
- Classification (분류)<지도>
- Clustering (군집)<비지도>

레이블 유무에 따라 지도, 비지도 학습으로 분류

### Regression

가격, 개수 등 예측

데이터를 `Training`할 부분, `Validation`할 부분으로 쪼갬.  
`Training`으로 학습시키고, `Validation`으로 검증함.

### Classification

여러 요인 -> 등급

Confidence Metrics (혼동 행렬)

### Clustering

데이터를 이용하여 그룹으로 나눔.

K-means Clustering
