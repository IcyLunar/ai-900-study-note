## 책임감있는 AI의 원칙

- `Fairness` (공정성)
- `Reliability & Safety` (신뢰성 및 안정성)
- `Privacy & Securiy` (보안 및 개인정보)
- `Inclusiveness` (포괄성): 장애인에 대한 배려
- `Transparency` (투명성): 모델에 대한 설명
- `Accountability` (책임감): 시스템이 내린 결정을 인간이 수정
